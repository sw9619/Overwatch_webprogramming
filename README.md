Overwatch Web Programming

오버워치 게임 정보를 제공하는 웹 애플리케이션입니다.
사용자가 영웅 정보, 게임 데이터 및 관련 콘텐츠를 쉽고 직관적으로 확인할 수 있도록 제작했습니다.

프로젝트 소개

본 프로젝트는 웹 프로그래밍 과목에서 진행한 프로젝트로,
프론트엔드 기술을 활용하여 사용자 친화적인 UI와 반응형 웹 페이지를 구현하는 것을 목표로 개발되었습니다.

Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Git & GitHub

주요 기능

- 메인 페이지 구현
- 영웅 정보 조회
- 반응형 웹 디자인
- 네비게이션 및 UI/UX 개선
- 게임 관련 콘텐츠 제공


- UI/UX 설계
- 프론트엔드 개발
- 페이지 레이아웃 구현
- GitHub를 활용한 형상 관리
