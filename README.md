# 스마트 소화기 관리 & 점검 애플리케이션

## 프로젝트 기획 의도 및 배경
교내 화재 경보 발생 당시, 대다수의 학생(약 86%)이 소화기의 위치를 정확히 알지 못한다는 문제점을 발견했습니다. 이를 해결하고 초기 화재 대응 능력을 강화하기 위해, 누구나 쉽게 주변의 소화기를 등록하고 상태를 직관적으로 관리할 수 있는 애플리케이션을 개발하게 되었습니다. 

단순한 목록 관리를 넘어, 아두이노와 블루투스(BLE) 통신을 연동하여 실제 소화기의 정상 작동 여부(소리 및 빛)를 원격으로 점검할 수 있는 하드웨어 제어 기능을 포함하고 있습니다.

## 주요 기능 

### 1. 소화기 등록 및 관리 
- **간편한 등록:** 첫 화면에서 카메라/갤러리를 통한 소화기 사진 첨부 및 이름 등록 기능.
- **스마트 목록 조회:** 등록된 소화기 리스트를 한눈에 파악. (등록된 소화기가 없을 경우 목록 진입을 차단하고 안내 메시지 출력)
- **삭제 기능:** 목록에서 불필요하거나 폐기된 소화기 정보를 간편하게 삭제.

### 2. 하드웨어 연동 원격 점검
- **블루투스(BLE) 페어링:** 상세 페이지 내 블루투스 연결 버튼을 통해 근거리 아두이노 기기와 통신.
- **비트마스크 기반 기기 제어:** - `소리 점검 버튼`: 비트 연산(Bit 0)을 통한 부저 제어.
  - `빛 점검 버튼`: 비트 연산(Bit 1)을 통한 LED 제어.
- **추가 사진 등록:** 초기 등록 시 사진을 누락했더라도 상세 페이지에서 언제든 추가 첨부 가능.

### 3. 직관적인 UI/UX
- 하단 고정 버튼 및 카드형 레이아웃을 채택하여, 토스(Toss) 앱처럼 깔끔하고 사용자 친화적인 경험 제공.

### 4. 보안 및 인증 (Security)
- JWT(JSON Web Token) 및 관리자 전용 인증 코드를 도입하여, 인가된 사용자만 소화기 데이터를 관리할 수 있도록 보안성 강화.

##  아두이노 연동 로직
앱과 아두이노는 BLE 통신을 통해 데이터를 주고받습니다. 앱에서 점검 버튼을 누르면 특정 비트 값을 전송하고, 아두이노는 이를 수신하여 하드웨어를 제어합니다.
- `0x01`: 소리(Buzzer) ON/OFF
- `0x02`: 빛(LED) ON/OFF
